# RyanCharts (Expense Tracker)

A modern, Vue 3-based expense tracking application designed to provide comprehensive financial insights for projects, clients, and employees. Built with Vuetify and Chart.js, it offers a premium user interface and detailed analytics.

## Features

- **Employee Financial Impact Chart**: Vertical analysis of employee contribution to project revenue vs. expenses using interactive charts.
- **Detailed Expense Tracking**: Monitor and categorize expenses by project, client, vs. non-project related costs.
- **Interactive Dashboards**: Visual data representation using `vue-chartjs` for easy interpretation of financial health.
- **Responsive Design**: Fully responsive layout optimized for various screen sizes using Vuetify and Tailwind CSS.
- **Modern Tech Stack**: Leveraging the power of Vue 3 (Composition API), Vite, Pinia for state management, and Tailwind CSS for utility-first styling.

## Tech Stack

- **Framework**: [Vue 3](https://vuejs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **UI Component Library**: [Vuetify 3](https://vuetifyjs.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Charting**: [Chart.js](https://www.chartjs.org/) & [vue-chartjs](https://vue-chartjs.org/)

## Project Setup

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/mrparrth/RyanCharts.git
   cd RyanCharts
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

### Development

To start the development server with hot-reload:

```sh
npm run dev
```

### Production Build

To build the application for production:

```sh
npm run build
```

To preview the production build locally:

```sh
npm run preview
```
